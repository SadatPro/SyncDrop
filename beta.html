🚀 Hybrid Transport Architecture

SyncDrop uses a dual-layer transport system for guaranteed connectivity:

wifi P2P Direct Mode

Fastest transfer directly between browsers. Works on 70-80% of connections.

cloud Cloud Relay Mode

Automatic fallback through Cloudflare's global network. 100% connectivity guarantee.

How it works: When you start a transfer, SyncDrop tries P2P first. If that fails within 15 seconds, it seamlessly switches to relay mode. Files are encrypted end-to-end and never stored on servers.
